# MoviesAndMe
Initiation à React Native

Application qui permet grâce a un mot-clef de trouver des titres de films contenant ce mot-clef.
Connexion avec une API https://www.themoviedb.org/ 
Projet terminé. 

Notions utilisées et apprises : 
- React Native
- Render
- SetState
- Props
- JSX
- FlexBox
- TouchableOpacity
- FlatList
